Note Preview Addin for Tomboy Notes - http://www.robpvn.net/notepreviewaddin

1) Installing

In order to install the add-in, simply place NotePreviewAddin.dll in your Tomboy plugins folder.

It is enabled by default, and can be turned off in Preferences -> Add-ins.

Your Tomboy plugins folder varies depending on your platform, see http://live.gnome.org/Tomboy/Directories.

2) Using

Using is very easy, hover your mouse cursor over a note link in the text and a tooltip will be displayed with a preview of the first 400 characters in the linked note.

3) Changelog

Version 0.10: Original release.

4) Compiling

The add-in can be compiled using make, or sudo make install to send it straight to the addin folder on Linux. Open the Makefile to see which compiler commands are used.
